android studio SQLite Database小例

2015年3月29日

##简述
SQLiteDatabase是一个可以进行增(Create)、查(Retrieve)、改(Update)、删(Delete)数据，即CRUD操作的类。
下面教程将向你展示如何使用SQLiteDatabase在Android中实现CRUD操作。

工具使用：
Android studio 1.1.0
